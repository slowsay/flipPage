# flipPage

翻页类 ，经常要用到列表数据，所以写了一个翻页插件

new flipPage('#viewcon', data, 6);

就这么简单！

功能
-----------------------------------------------------
*前一页flipPage.toFront
*后一页flipPage.toBack
*列表 flipPage.render
*隔列颜色 flipPage.lineBackground
*页数 flipPage.pageSize
