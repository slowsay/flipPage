# flipPage

翻页类 ，经常要用到列表数据，所以写了一个翻页插件<br />
<pre>
new flipPage('#viewcon', data, 6,{title:['标题0','标题1','标题2']});<br />
</pre>
就这么简单！<br />

功能
-----------------------------------------------------
*前一页flipPage.toFront<br />
*后一页flipPage.toBack<br />
*列表 flipPage.render<br />
*隔列颜色 flipPage.lineBackground<br />
*页数 flipPage.pageSize<br />
